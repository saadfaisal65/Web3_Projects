# 0xInnovations Hub

Welcome to **0xInnovations Hub**, your central gateway to advanced decentralized applications (dApps). Our platform is designed to combine seamless Web3 experiences with powerful functionality, offering a suite of decentralized tools for developers and users alike.

## 🚀 Projects within the Hub

The hub serves as a launchpad for the following projects:

### 🟢 Active Projects
- **[Nexus Faucet Hub](https://0xinnovations-testnet-distributer.vercel.app/)**: A premium multi-chain testnet token distributor for verified developers. Request tokens across various testnets easily.
- **[Decentralized Voting System](https://0xinnovation-decentralize-voting-system.vercel.app/)**: A secure, transparent, and immutable voting system powered by smart contracts on the blockchain.

### 🟡 Coming Soon
- **DeFi Dashboard**: Track your yields, monitor liquidity pools, and manage your decentralized portfolio securely.
- **Token Exchange**: Swap testnet tokens seamlessly with our high-speed, low-slippage decentralized exchange protocol.
- **Smart Contracts**: A secure playground to deploy, verify, and interact with smart contracts on various EVM networks.

## 🛠️ Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites
Make sure you have Node.js and npm (or pnpm/yarn) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone <https://github.com/saadfaisal65/0xinnovations.git>
   cd 0xinnovation-hub
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Setup environment variables (if any):
   Create a `.env.local` file in the root directory based on the configuration needed.

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000` to see the hub!
